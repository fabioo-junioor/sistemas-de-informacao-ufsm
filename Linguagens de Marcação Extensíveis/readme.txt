Programas a serem instalados
1- wampserver (nesse caso é para sistemas windows).

Após a instalação do wamp, você podera utiliza-lo como um servidor local,
é assim que podemos vizualizar no navegador scripts em php/html/css etc.

Quando instalado o wamp irá criar uma pasta no no disco C, com o nome 'wamp64/www',
qualquer script 'php/html/css' que for colocado dentro dessa pasta 'www' podera ser
vizualizado no navegador.

Exemplo: Se for criado uma pasta com nome 'site' dentro de 'www' é dessa forma que acessamos no
navegador: localhost/site/
